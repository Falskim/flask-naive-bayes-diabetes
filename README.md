# Flask Naive Bayes Diabetes

Early Stage Diabetes Risk Prediction Using Naive Bayes Classifier

## Installation

Install with pip:

```
$ pip install -r requirements.txt
```

## Running Apps
Make sure you already have `Flask` installed

```
$ flask run
```

Flask default port is 
`http://127.0.0.1:5000/`
